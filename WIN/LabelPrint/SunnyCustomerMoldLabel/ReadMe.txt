NiceLabel SDK.NET SimplePrintSample Project

What is it?
-----------

The NiceLabel SDK.NET SimplePrintSample is a C# WinForms project that 
demonstrates a simplistic use of the SDK for printing a label.


Key Examples
----------
The following functions are demonstrated in this sample:

Loading the SDK.NET assembly resources from an external directory.
Initalizing the print engine.
Opening a label file.
Setting values for label variables.
Setting the quantity of labels to be printed.
Executing the print method.


Highlights
----------

Program.cs -    The GetSdkAssembly method demonstrates loading the SDK.NET 
                assembly resources from an external directory.

Form1.cs -      The PrintButton_Click method demonstrates working with a 
                Label object.
